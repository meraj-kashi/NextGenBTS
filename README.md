# NextGenBTS
A demo for training Docker and K8S 

# Development  
Setup virtual environment:  
python3 -m pip install --user virtualenv  
python3 -m venv .venv  
Note: sudo apt install python3.10-venv  
source .venv/bin/activate
pip3 install -r requirements.txt
